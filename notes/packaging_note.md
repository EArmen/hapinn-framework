# Packaging note

This GitHub-ready package was assembled from the provided manuscript source, compiled PDF, and repository scaffold notes.

## Included source files

- `paper/main.tex` from the provided LaTeX source
- `paper/main.pdf` from the provided compiled PDF
- `paper/references.bib` as a convenience export generated from the inline bibliography already present in `main.tex`

## Important note

The manuscript source currently contains its own `thebibliography` block, so `references.bib` is included for convenience and future cleanup, not because the current source depends on it.
