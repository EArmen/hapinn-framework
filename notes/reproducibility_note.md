# Reproducibility Note

This repository accompanies a framework paper on HAPINN.

## What is included

The repository includes:

- the LaTeX source of the manuscript;
- the compiled PDF of the paper;
- supporting notes and supplementary text materials.

## What is not included

This repository does not provide a new benchmark software package or a new large-scale experimental suite.

That is intentional and consistent with the article itself. The associated manuscript is not an empirical benchmark paper. Its contribution is conceptual, methodological, and theoretical.

## Empirical scope

The paper discusses a previously published semiconductor realization as an application-level instance of the HAPINN framework. Those results are cited and interpreted in the manuscript, but they are not introduced in the repository as newly generated results.

## Purpose

The purpose of this repository is to support transparent scholarly communication and accurate documentation of the manuscript.
