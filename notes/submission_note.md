# Submission Note

This repository corresponds to the manuscript:

**HAPINN: A Solver-Guided Anchor-Based Framework for Parameterized Differential Equations**

## Intended role of the repository

The repository is a paper companion repository. Its primary goal is to make the manuscript source and supporting notes available in a clean and transparent form.

## Important scope statement

The associated article is a framework paper. It is not intended to serve as a new benchmark study, and it does not claim a new large-scale empirical campaign.

## Recommended contents for submission stage

At minimum, the repository should contain:

- `paper/main.tex`
- `paper/main.pdf`
- bibliography file, if applicable
- `README.md`
- `CITATION.cff`
- `LICENSE`

## Suggested maintenance policy

If the article is revised after peer review, the repository should be updated so that the LaTeX source, PDF, and citation information remain synchronized with the submitted or accepted manuscript version.
